# UPC
Usb Power Controller

USB電源を制御する基板のためのプログラムです。<br>
マイコンはXIAO-RP2040<br>
言語はMicroPython<br>

機能概要<br>
・USB-microB電源をつないで、USB-Aに出力<br>
・つないでから設定時間後にON<br>
・つなぐとON、設定時間後にOFF<br>
・指定時間ごとにONを繰り返す<br>
等の動作ができます。<br>
BEC機能(バッテリー省電力機能キャンセル)もあります。<br>
