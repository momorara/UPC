# UPC
Usb Power Controller

USB電源を制御する基板のためのプログラムです。<br>
マイコンはXIAO-RP2040<br>
言語はMicroPython<br>

機能概要<br>
・USB-microB電源をつないで、USB-Aに出力<br>
・つないでから設定時間後にON<br>
・つなぐとON、設定時間後にOFF<br>
・指定時間ごとにONを繰り返す<br>
等の動作ができます。<br>
BEC機能(バッテリー省電力機能キャンセル)もあります。<br>
1日に何回もon-offを繰り返すような動作では誤差が大きくなります。<br>
・誤差補正機能<br>
誤差補正前では日差10秒前後の誤差がありますが、誤差補正機能があり調整すれば、月差1,2秒程度まで補正することが可能です。補正機能の使い方などは取扱説明書を確認ください。<br>
